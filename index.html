Sobre mí

"Soy una artista plástica y programadora apasionada, con una personalidad amigable y sociable. Mi vida está centrada en la creatividad y el pensamiento intelectual. Constantemente activa y en búsqueda de nuevos retos, combino mi habilidad artística con mi pasión por la programación para crear proyectos innovadores y visualmente impactantes. Proyectos

Lobo patagonico

En el lienzo se presenta un lobo patagónico, destacando en su hábitat natural de la vasta estepa. El lobo, con su pelaje gris plateado y ojos amarillos serenos, está retratado en una pose erguida, observando tranquilamente el horizonte. El fondo del cuadro muestra las suaves colina, pintados con tonos cálidos repleto de flores amarrillas. El artista ha capturado la textura del pelaje del lobo y la sensación de calma en el paisaje, transmitiendo la serenidad y la belleza de esta región remota.

Habilidades

Desarrollo Web Frontend

Creación de interfaces de usuario interactivas y responsivas utilizando HTML, CSS y JavaScript. Experiencia con frameworks modernos como React o Vue para desarrollar aplicaciones web dinámicas.

Diseño UX/UI

Diseño intuitivo y atractivo centrado en la experiencia del usuario (UX). Creación de interfaces de usuario (UI) que combinan estética visual con funcionalidad para mejorar la usabilidad y satisfacción del usuario.

Desarrollo Backend

Construcción y gestión de la lógica y la base de datos detrás de las aplicaciones web. Uso de tecnologías como Node.js y Python con frameworks como Express o Django para crear sistemas robustos y escalables.

Diseño Gráfico

Creación de identidades visuales cohesivas y atractivas para marcas y proyectos. Experiencia en herramientas como Adobe Illustrator y Photoshop para diseño de logotipos, material publicitario y contenido digital.

SEO y Marketing Digital

Implementación de estrategias de optimización para motores de búsqueda (SEO) para mejorar el posicionamiento y visibilidad en línea. Conocimientos en análisis de datos, campañas de marketing digital y gestión de redes sociales.

Desarrollo de Aplicaciones Móviles

Creación de aplicaciones móviles nativas e híbridas para iOS y Android utilizando herramientas como Flutter, React Native o Swift. Enfoque en la usabilidad móvil y la optimización de rendimiento.

Gestión de Proyectos

Coordinación efectiva de equipos y recursos para cumplir con objetivos de proyectos dentro de plazos y presupuestos establecidos. Uso de metodologías ágiles como Scrum o Kanban para mejorar la productividad y la calidad del producto final.

Analítica y Visualización de Datos

Extracción, limpieza, análisis y visualización de datos utilizando herramientas como Python (pandas, matplotlib) o R. Creación de informes y paneles interactivos para facilitar la toma de decisiones basadas en datos.
